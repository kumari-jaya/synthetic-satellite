Installation Guide

Prerequisites

  • Node.js v18 or higher
  • npm v9+ or yarn v4+
  • Minimum 4GB RAM
  • Compatible with Windows, macOS, and Linux

Basic Installation


# Using npm
npm install @vortx/ai-core @vortx/ai-plugins

# Using yarn
yarn add @vortx/ai-core @vortx/ai-plugins

# Using pnpm
pnpm add @vortx/ai-core @vortx/ai-plugins

Quick Start


import { VortxAI } from '@vortx/ai-core';
import { ImageAnalysis } from '@vortx/ai-plugins';

const vortx = new VortxAI({
  apiKey: process.env.VORTX_API_KEY,
  version: '2025.1',
  plugins: [ImageAnalysis]
});
